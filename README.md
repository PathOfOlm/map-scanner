# Map Scanner

A Meteor Client addon that scans nearby item-frame maps and reconstructs them as images.

Maps can optionally be sent to a Discord webhook, configurable directly from the Meteor GUI.  

---

## Features

- Scans nearby item frames containing filled maps
- Reconstructs 128Ã—128 map images
- Discord webhook output
- Avoids re-sending the same map multiple times

---

## Installation

1. Download the latest `.jar` from the **Releases** page
2. Place the jar into your Minecraft `mods` folder
3. Launch Minecraft with **Meteor Client**
